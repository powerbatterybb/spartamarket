# SpartaMarket
---

작성자: 지효상

### 사이트 개요
---

목적: 중고물품을 사고 파는 게시판
기능: 게시글 CRUD, 댓글 CRUD, Riot API 이용 실시간 최상위권 유저 조회, 회원가입

### 개발환경
---
기술스택: Django, HTML, CSS

### 개발기간
---
2024/04/15 ~ 2024/04/19

### 기능설명
---
1. Create 
    * <post_register.html>
    * 글/댓글 작성 한 것을 게시판에 추가해 주는 기능


2. Read
    * <post_detail.html>
    * 글 작성 한 것을 게시판에서 상세페이지로 이동해 보여주는 기능
    * 글 상세 페이지에 댓글 보여주는 기능

    
3. Update
    * <post_edit.html>
    * 글/댓글을 수정하는 기능


4. Delete  
    * <index.html>
    * 글/댓글을 삭제하는 기능

5. Signup
    * <signup.html>
    * 회원가입하는 기능
